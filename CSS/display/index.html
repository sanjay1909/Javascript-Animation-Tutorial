CSS - Display

Every element on a page is a rectangular box. Behavior of this box is determined by the CSS display property. Elements are classified as Block or Inline elements based on display property. Block elements are considered as container, they can hold block and inline elements.Inline elements can hold another inlin element but cannot hold a block level Element.
