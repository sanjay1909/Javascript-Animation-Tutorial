Positioning

The CSS positioning properties[top, bottom, left, and right] allow you to position an element. However, these properties will not work unless the position property is set first. They also work differently depending on the positioning method. There are 4 positioning methods.

  • Static (Default)
  • Fixed (Based on window)
  • Relative (Based on its own Position)
  • Absolute (Based on its parent, whose position is not static)
