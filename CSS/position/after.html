CSS - Position

Div1[Block]: Relative... Span1[inline]:Absolute - "positioned based on first parent [Div1] that has a position other than static. If no such element is found, the containing block is HTML". In this case its postioned based on Div1(relative) by top: [16px] and left: [10px]
Div2[Block]: Static... Span2[inline]:Absolute - "positioned based on first parent [Div1] that has a position other than static. If no such element is found, the containing block is HTML". In this case ancestoral parent Div2 - static, so instead of appearing iniside Div2, Span2 appears based on HTML block by top: [0px] and left: [60px]
Div3[Block]: Static... Span3[inline]: Fixed - positioned based on browser window, and will not move even if the window is scrolled. In this case moved to top: [140px] and right: [5px] based on windows position
Div4[Block]: Static... Span4[inline]: Relative - positioned based on its normal position. In this case moved to top: [-10px]and left: [20px] from its normal position
